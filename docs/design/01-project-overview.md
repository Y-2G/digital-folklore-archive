# プロジェクト概要

## 目的

- ネット由来を含む現代の伝承（怪談・都市伝説・クリーピーパスタ等）を「資料」として収蔵し、出典と変遷（版履歴）を添えて公開する
- 演出（ノイズ/グリッチ等）によらず、**情報設計と読書体験**で価値を出す

## トーン

- 完全に資料館（UIで怖がらせない）
- 罫線・余白・規則正しいグリッド
- 目録（行リスト）主体、カードUIを避ける

## 特徴

| 項目 | 方針 |
|------|------|
| UI | 目録型（図書館UI） |
| 演出 | ゼロ（0） |
| 言語 | 多言語（JP/EN） |
| 収蔵対象 | 怪談・都市伝説・クリーピーパスタ・創作・解説 |
