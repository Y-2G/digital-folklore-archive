# 多言語対応（i18n）

## 方針

- UI言語: JP/EN切替
- ラベル（Type/Confidence）は英語固定でも運用可（資料庫トーン＋国際性）
- next-intl を使用

## 対応言語

| コード | 言語 |
|--------|------|
| `ja` | 日本語 |
| `en` | 英語 |
